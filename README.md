# Welkin Technohub PVT.LTD — Company Website

Official website for **Welkin Technohub PVT.LTD**, an IT services company based in Ahmedabad, Gujarat, India.

## About

A multi-page marketing/brochure website presenting the company's services, technology expertise, and a contact/career inquiry form.

## Pages

- **Home** — Hero, About, Services, Technologies overview
- **About** — Company philosophy and team
- **Services** — Web Development, App Development, UI/UX Design, QA Services
- **Technologies** — PHP, .Net, iOS, Android, Angular.js, React.js, HTML5, MongoDB, MySQL, and more
- **Projects** — Portfolio of completed work
- **Contact** — Address, Google Maps embed, and inquiry/career form

## Tech Stack

- HTML5, Bootstrap 4, jQuery
- AOS (scroll animations), Isotope (filtering), Owl Carousel
- PHP contact form (`forms/contact.php`)
- Google Fonts, IcoFont, Boxicons
